수집된 데이터 정리 => 전처리
1) 빈칸/데이터 없는 것 검사 => 결측치 Missing value
    => 재수집
    => 일부 미미하게 존재 ==> 처리 : 버리기/치환
2) 중복 데이터(행) 검사
    => 학생이름, 학교, 학년, 반, 번호 모두가 같은 데이터 -> 불가능, 제거해야됨
    => 날씨데이터, 날짜, 온도, 습도 --> 중복 가능/ 유지
3) 이상한 값 즉, 이상치/특이값 검사
    => 컬럼별로 유효한 값의 범위
    => (예: 성별 - 남,여/ 나이 -1, 130)
    => 검사/처리 => 삭제/치환
    => 테이터가 가진 특성 체크 꼭!!
